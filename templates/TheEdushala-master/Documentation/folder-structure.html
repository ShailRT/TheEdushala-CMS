  • Edomi - Online Education & Learning Courses HTML CSS Responsive Template / : Root template folder contain all html, js, css, scss, images and other files.
  • assets/ : Folder contain all the listing Template assets which has css, js, scss, and images of LTR version.
    • css/: Folder contain assets which has sample scss, css, js files.
      • admin-custom : admin-custom style sheets for admin Template.
      • sidemenu : sidemenu style sheets for Sidemenu.
      • icons : icons.css link has imported in all HTML Pgaes.
      • style : Main style sheet for template.
    • css-rtl/: Folder contain assets which has sample scss, css, js files and images of RTL version.
      • admin-custom : admin-custom style sheets for admin Template.
      • sidemenu : sidemenu style sheets for Sidemenu.
      • icons : icons.css link has imported in all HTML Pgaes.
      • style : Main style sheet for template.
    • iconfonts/ : Template icons lists.
    • images/ : Template images.
    • js/ : Folder contain all the template pages js files.
    • plugins/ : Folder contain all template plugins.
    • scss/ : Folder contain all pages scss files of LTR version.
    • scss-rtl/ : Folder contain all pages scss files of RTl version.
    • video/ : Template Videos.
  • gulpfile.js : Gulp tasks js file.
  • package.gulp.json : Gulp package json file.
